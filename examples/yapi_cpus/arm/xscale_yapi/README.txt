This is the xscale model using the yapi platform to represent pipelined slot delays.

Files used:
=============

Makefiles:
----------------
Makefile               - top level make file
Makefile_mmscs         - makefile generated by sim (not used)
Makefile_mmscs_xscale  - modified makefile from sim (used)
Makefile_mmscs_xscale_debug - modified makefile from sim for debug (used)
Makefile.newsim        - for the new sim (same as Makefile)
Makefile_old           - old makefile for old simulator
Makefile.oldsim        - old makefile for old simulator

Supporting .cpp and .h files used for interpreting traces: (modified from the SWARM ISS)
----------------------------
alu.cpp
alu.h
disarm.cpp
disarm.h
isa_be.h
isa.h
isa_le.h
swarm.cpp
swarm.h
swarm_macros.h
swarm_types.h
trace.cpp
trace.h
tracesim.cpp
tracesim.h

Metropolis Files:
-------------------
xscale_netlist.mmm - the toplevel netlist
Exec.mmm           - handles execution delays + dependencies of instructions    
Fetch.mmm          - inputs trace file, does branch prediction, issues 
instructions
MyDataTypes.mmm    - Data types used, such as the instruction type
XScaleExec.mmm     - A wrapper around Exec parameterized for the XScale
XScaleFetch.mmm    - A wrapper around Fetch parameterized for the XScale
