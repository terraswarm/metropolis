This is a test case for testing the runtime library located under 
metro/src/metropolis/metamodel/runtime and the features of the elaborated
network.

To run the test case, under the current directory, type "make" to 
compile the meta-model and print out the testing information.

RuntimeTestBackend will be invoked to access the elaborated network,
and test various features of the runtime library and elaboration.
